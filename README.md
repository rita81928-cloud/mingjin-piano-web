# 謝明瑾 個人網站與 AI 開發專案 (Ming-Chin Hsieh Portfolio)

這是一個由前端技術 (HTML, CSS, JavaScript) 結合現代化網頁設計美學（如 Glassmorphism 琉璃擬物特效與平滑滾動動畫）所建置的綜合型網頁專案。
本專案分為兩個主要部分：**個人靜態網頁介紹** 以及 **AI 輔助開發之幼兒互動鋼琴遊戲**。

## 目錄結構 (Directory Structure)

```text
portfolio/
├── about-me/        # 謝明瑾 個人形象與經歷介紹網頁
│   ├── index.html   # 主入口文件
│   ├── style.css    # 客製化深色質感主題與特效排版
│   ├── script.js    # 捲動淡入動畫與自訂游標特效
│   └── profile.jpg  # 個人形象照
│
└── piano-kids/      # 結合 AI 技術開發的互動式「Magic Piano」遊戲網頁
    ├── index.html   # 遊戲主頁與雙八度鋼琴鍵盤結構
    ├── style.css    # 遊戲介面視覺設計
    └── script.js    # 鋼琴發聲邏輯與遊戲計分互動腳本
```

## 專案功能與特色 (Features)

### 1. 關於我 (About Me)
- **深色時尚美學**：採用沉穩的深色背景搭配奢華金色點綴（`#d4af37`），反映古典音樂領域的高雅氣質。
- **響應式設計 (RWD)**：支援桌機與手機瀏覽，經歷卡片與榮譽榜單在小螢幕上會自動垂直排列，確保最佳閱讀體驗。
- **動態互動效果**：當使用者下拉頁面時，卡片會自動淡入；畫面中並加入了自訂的光暈跟隨游標效果。
- **身分轉換與雙向連結**：展示「鋼琴演奏博士」與「AI智能體中級工程師」雙重身分，並可透過右上角導覽列直接跳轉至 AI 開發的互動作品專案。

### 2. 幼兒鋼琴教學遊戲 (Piano Kids - Magic Piano)
- **AI 智能體獨立開發**：本互動網頁遊戲是完全由 AI 智能體（AI Agent）運用程式編寫技術輔助開發完成的作品。
- **鍵盤雙八度敲擊**：支援在電腦鍵盤上按下對應按鍵（如 Z, X, C 或 Q, W, E）來觸發真實音階發聲。
- **記憶訓練模式**：內建記憶與反應力訓練的遊戲機制，提高幼兒學習鋼琴的互動性與趣味性。
